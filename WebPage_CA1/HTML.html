Introduction to HTML

HTML stands for

HTML stands for HyperText Markup Language. HTML is the combination of Hypertext and Markup language. Hypertext defines the link between the web pages. The markup language is used to define the text document within a tag that defines the structure of web pages.

What HTML CODE does?

  • TML code allows us to build websites
  • HTML describes the structure of a web page
  • HTML response images on a web pages
  • HTML provides the Internet navigation on a web site

HTML Elements

HTML uses elements (tags) to manage a structure of a web page content. By using the tags we can establish headings, paragraphs, text, lists, inks, images, tables, etc.

Two types of elements:

  • Container element (< p>Pragraph< /p>; < h1> Heading1< /h1>)
  • Stand alone element ( < img>Pic1< /img>)
