Introduction to CSS

CSS stads for

CSS stands for Cascading Style Sheets. It is the language to style web pages including colors, layout, and fonts. It allows one to adapt different types of devices, such as large and small screens. CSS styles web pages by interacting with HTML tags. CSS is independent of HTML and can be used with any XML-based markup language.

What CSS does ?

  • CSS control the layout of multiple web pages
  • CSS store external stylesheets in a file
  • CSS response images on a web pages
  • CSS shows us how HTML tags represented on
    a screen of digital devices

CSS connection with HTML

CSS has three different ways to connect with HTML pages. Inline and Internal
do not use much as it is not far efficient. The most efficient way is an external connection as it allows us to use the same code to style multiple numbers of
web pages. It reduces the time to download subsequent pages.

CSS selectors

A CSS selector is used for applying color, text, etc to HTML tags that developers want to style. There are many of them. The page represented the most useful.

Types of selectors
  • Universal *{};
  • Type h1, h2, h3 {};
  • Class .class {} / p.class{}
  • ID #intro {}
ID and Classes

CSS selector also separates Id and classes: Id styles the specific tag and belongs only to the tag. Classes are much more flexible. We can apply the same class (color or text) to many numbers of tags. It helps developers to create the same style of elements.

Examples:

  • ID selector: #footer{property: value};
  • Class selector: .class {property: value}
What we can build by using selectors

We can build boxes that wrap around every HTML element. It consists of: margins, borders, padding, and the actual content. By color property to set the color of the text, borders and background. The background of the page or boxes can be a picture or photo that is linked to an HTML file. By using font property can manage the sizes and style of the text. Also by using CSS we can make an animation of some objects.
