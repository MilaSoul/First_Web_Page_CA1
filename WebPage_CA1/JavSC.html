Introduction to JavaScript

JavaScript stand for

JavaScript stands for WEB/HTML scripting language. JavaScript has been created to make web pages alive. JavaScript runs on the client-side of the web, which can be used to design / program how the web pages behave on the occurrence of an event.

Functionality

To define a function which used in various HTML documents it's better to keep that functionality in a separate JavaScript file and then include that file in your HTML documents. A JavaScript file has an extension as .js and it is included in HTML files using < scrpt > tag. JavaScript code can go inside of the document < head > section to keep them contained and out of the main content of the HTML document. We can call the code from the file by Functions.
